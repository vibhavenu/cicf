# Project Management System

A simple and responsive **Project Management System** developed using **HTML, CSS, and JavaScript**. The application allows users to securely log in, browse projects, search and filter projects by status, and view detailed information about each project through a clean dashboard interface. :contentReference[oaicite:0]{index=0}

## Features

- Secure login screen with demo credentials
- Responsive project dashboard
- Search projects by project name or client
- Filter projects by status
- View project details in a popup modal
- Displays project information including:
  - Client
  - Due date
  - Status
  - Description
  - Technology stack
  - Team members
- Logout functionality
- Responsive design for desktop and mobile devices

## Technologies Used

- HTML5

## How to Run

### Method 1

1. Download or clone the repository.
2. Open the project folder.
3. Open `dashboard.html` in any modern web browser.

### Method 2 (Recommended)

1. Open the project in Visual Studio Code.
2. Install the **Live Server** extension.
3. Right-click `dashboard.html`.
4. Select "Open with Live Server".
